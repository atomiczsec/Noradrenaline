# os_version

Collects macOS and Linux operating system version details: product, version, kernel, architecture, and install-age evidence.

## Exports

| Function | Arguments | Description |
|----------|-----------|-------------|
| `os_version` | none | Reports product, version, kernel, architecture, and install-age evidence. |

## Example Output

```text
[+] os_version
    Product:    macOS (15.5)
    Build:      24F74
    Edition:    macOS / Client
    Kernel:     Darwin 24.5.0
    Arch:       arm64
    InstallDate: 1714690000 (Unix Timestamp, from /var/db/.AppleSetupDone)
```
